Aqui está a demonstração de como integrar o Plugin OpenPix em seu aplicativo de frontend. O que temos nesta demonstração:
- gera uma cobrança com valor de 1 centavo
- recebe eventos do plugin como status de pagamento
- mostra automaticamente quando a cobrança foi paga

[React Plugin Demo](https://github.com/Open-Pix/react-openpix-plugin-demo)

Você também pode testar o Plugin usando nosso [Online Demo](https://openpix.com.br/plugin/).
