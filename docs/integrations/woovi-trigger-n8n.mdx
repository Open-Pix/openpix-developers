## Documentação

1. Crie uma API na Woovi [aqui](../apis/api-getting-started.md#passo-1---criar-um-novo-aplicativo)
2. Copie o AppID
3. Acesse a sua instância do N8N
4. Acesse `Configurações` > `Community nodes`
4.1 Clique em `Install` e pesquise por `n8n-nodes-woovi`
![Woovi Trigger](/img/integrations/n8n/install-node.png)

5. Crie um novo Workflow e utilize o nó `Woovi Trigger`

![Woovi Trigger](/img/integrations/n8n/select-woovi-trigger.png)
