Para criar clientes em lote acesse o menu de [Clientes](https://developers.woovi.com/docs/payment/payment-how-to-request-access) 
na plataforma e clique no opção de **Cliente em Lote**, presente no canto direito superior.

Você será redirecionado para a página de [Cliente em Lote](https://app.woovi.com/home/crm/create-batch), onde encontrará os passos
para criar clientes em lote.

No primeiro passo você encontra os campos para preencher na planilha para importação. Você pode tambem baixar o nosso **modelo do excel**, 
uma planilha excel que ja possui os campos em ordem para facilitar o preenchimento dos dados.

Após ter a planilha com os dados preenchidos, você pode enviar ele no campo de upload, selecionando o arquivo ou arrastando e soltando para dentro da área selecionável.

Após isso, basta clicar em `Avançar` para ir pro próximo passo.

![area-for-upload-file](/img/customer/batch/area-for-upload-file.png)

Após avançar pro próximo passo você poderá verificar os dados enviados em uma tabela, onde poderá editá-los dando um clique duplo no campo desejado. 

Caso algum campo esteja com um formato diferente do correto, um erro sobre a linha poderá ser encontrado, basta clicar para expandir para ter mais informações sobre o erro.

![batch-with-error](/img/customer/batch/batch-with-error.png)

Após validar e corrigir (caso necessário) todos os dados, clique em **Registrar XX Clientes**.

Os clientes serão registrados e o resultado aparecerá na sua tela.

Você encontrará os clientes registrados, e poderá navegar até a lista de clientes clicando no botão **Ver Clientes**.

![go-to-customer-list](/img/customer/batch/go-to-customer-list.png)
