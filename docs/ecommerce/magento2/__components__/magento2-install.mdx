### Instalação via Composer

É possível instalar o nosso plugin utilizando o Composer.

Realize os seguintes passos:
1. Acessar Magento Marketplace
2. Buscar pelo Plugin OpenPix
3. Realizar a compra grátis do mesmo
4. Clicar em instalar
5. Em "Meus Downloads" siga as instruções a direita

![Instructions](./__assets__/composer-instructions.png)

6. Um exemplo de comando é `composer require openpix/pix`. Acesse também a [documentação do Magento2](https://experienceleague.adobe.com/docs/commerce-operations/installation-guide/tutorials/extensions.html) para mais detalhes.
